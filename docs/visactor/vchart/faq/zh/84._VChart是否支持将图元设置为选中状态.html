问题标题

VChart是否支持将图元设置为选中状态？

问题描述

可以在spec中设置让某个点在不交互时始终处于选中状态吗？

解决方案

直接接通过 spec 设置，不走 state 接口的话可以直接通过通道的自定义函数来设置样式，参数中包含数据信息，根据不同的数据来区分图元是否需要设置为选中状态即可。
