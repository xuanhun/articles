问题标题

如何限制Tooltip的行数？

问题描述

Tooltip中数据条数非常多的时候，超过20条后会自动归纳为其他，这个数量支持自定义吗？

解决方案

tooltip支持通过 maxLineCount设置最大行数，需要注意的是，vchart的tooltip支持多种类型，现在需要分别设置；

代码示例

const spec = {
  type: 'line',
  data: {

    mark: {
      maxLineCount: 2
    },
    dimension: {
      maxLineCount: 2
    }
