问题标题

VChart 中如何单独配置图例的样式，改变图形的形状

问题描述

如何将柱图一个系列的图例项图形改成圆

解决方案

VChart 中图例可以通过data配置项自定义，其中图形属性在图例项的 shape 属性中

 legends: {
    visible: true,
    data: items => {
      return items.map(item => {
        if(item.label === 'Under 5 Years'){
          item.shape.symbolType = 'circle';
        }
        return item;
      });
    },
  },</br>
