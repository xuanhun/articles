问题标题

hover 超长label时候上方显示的这个tip能否配置修改背景颜色和文字颜色

问题描述

hover 超长label时候上方显示的这个tip能否配置修改背景颜色和文字颜色

解决方案

在主题中配置 poptip 属性即可。

theme:{
    component: {
      poptip: {
        contentStyle: {
          fill: '#fff',
        },
        panel: {
          fill: '#ccc'
        }
      }
    }
  }</br>
