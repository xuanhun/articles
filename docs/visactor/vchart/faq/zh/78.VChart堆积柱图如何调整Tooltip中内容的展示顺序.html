问题标题

VChart堆积柱图如何调整Tooltip中内容的展示顺序？

问题描述

堆叠柱状图中，堆积的顺序好像和数据的顺序相反一样，然后 tooltip 又反过来，要怎么让柱子的颜色顺序和 tooltip 的顺序一致？

解决方案

对于堆积柱状图，可以有两个方案，使得tooltip中内容的顺序和柱子的堆积顺序一致：
* 方案一：stackInverse:true调整堆积的顺序

{
  stack: true,
  stackInverse: true
}</br>

  • 方案二：调整tooltip中元素的顺序
tooltip: {
    dimension: {
      updateContent: (items) => {
        return items.reverse();
      }
    }
  }</br>
