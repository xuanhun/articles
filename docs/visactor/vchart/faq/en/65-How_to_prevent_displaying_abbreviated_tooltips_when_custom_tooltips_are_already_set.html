Title

How to disable the display of abbreviated tooltips. Customized tooltips do not need to be displayed again?

Description

How to disable the display of abbreviated tooltips? Customized tooltips do not need to be displayed repeatedly, as shown below:

Solution

VChart will automatically omit long text and provide an interactive floating display poptip to display the complete text by default. If you want to turn it off, you can set poptip: false in the parameter of the constructor function when initializing the chart, as follows:

const vchart = new VChart(spec, { 
  // ...
  poptip: false, // Close poptip for omitted text
});</br>
