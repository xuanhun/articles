Question title

How to adjust the display order of content in Tooltip of VChartStacked Bar Chart?

Problem Description

In a stacked bar chart, the stacking order seems to be reversed from the order of the data, and then the tooltip is reversed again. How can I make the color order of the bars consistent with the order of the tooltip?

Solution

For stacked bar charts, there are two solutions to make the order of the content in the tooltip consistent with the stacking order of the bars:
* Solution 1: stackInverse:true to adjust the stacking order

{
  stack: true,
  stackInverse: true
}</br>

  • Solution 2: Adjust the order of elements in the tooltip
tooltip: {
    dimension: {
      updateContent: (items) => {
        return items.reverse();
      }
    }
  }</br>
