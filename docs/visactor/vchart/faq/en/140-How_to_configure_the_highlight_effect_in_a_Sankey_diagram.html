Title

How to configure the highlight effect of Sankey diagram?

Description

I added the following configuration based on this demo (see Figure 1), but it did not achieve the effect of Figure 2. How should I configure it?

Solution

If you want to achieve the highlighting effect in Figure 2, you need to configure the interaction states of node and link under the premise of emphasis. The above figure involves two interaction states: selected and blur, as follows:
